# Kinopoisk Redirect Button

Расширение для Chrome, добавляющее кнопку быстрого перехода со страницы фильма или сериала к его просмотру. Заменяет `kino` → `ss` в URL одним кликом.

## Установка

1. Скачай ZIP-архив: зелёная кнопка **Code → Download ZIP**
2. Распакуй в любую папку
3. Открой `chrome://extensions/` в Chrome
4. Включи **Режим разработчика** (справа сверху)
5. Нажми **Загрузить распакованное расширение** и выбери папку
6. Готово! Кнопка появится на всех страницах Кинопоиска справа снизу

## Как использовать

Зайди на страницу любого фильма или сериала на Кинопоиске и нажми круглую кнопку с иконкой — ты перейдёшь к просмотру. Та же страница, другой сайт.

# Kinopoisk Redirect Button

Chrome extension that adds a quick button to go from a movie or series page straight to watching it. Replaces `kino` → `ss` in the URL with one click.

## Installation

1. Download ZIP: green button **Code → Download ZIP**
2. Extract to any folder
3. Open `chrome://extensions/` in Chrome
4. Enable **Developer mode** (top right)
5. Click **Load unpacked** and select the folder
6. Done! A button will appear on all Kinopoisk pages, bottom right

## How to use

Open any movie or series page on Kinopoisk and click the round icon button — you'll jump straight to watching. Same page, different site.
